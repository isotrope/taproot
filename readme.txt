# Taproot WordPress Theme

Taproot is a modern, lightweight, yet powerful mutipurpose WordPress theme built on top of the Mythic theme framework and the Rootstrap
toolkit. 

## Installation

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Upload zipped file. 
3. Activate.
